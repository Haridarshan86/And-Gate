# AND Gate using Verilog HDL

A simple implementation of a 2-input AND Gate using Verilog HDL along with a testbench for simulation and verification.

An AND gate is a basic digital logic gate used in digital electronics and computer systems.

The AND gate performs a logical multiplication operation. It gives an output of HIGH (1) only when all the inputs are HIGH (1). If any input is LOW (0), the output becomes LOW (0).

📌 Working Principle

For a 2-input AND gate:

If A = 1 and B = 1 → Output Y = 1
Otherwise → Output Y = 0
